Welcome to Algernon

The Algernon rule-based inference system is now implemented in Java and interfaced with Protégé. Algernon performs forward and backward rule-based processing of frame-based knowledge bases, and efficiently stores and retrieves information in ontologies and knowledge bases.

This is Algernon version 5.0.1, compiled for Protege 2.1 and higher.

For more information, see the Algernon home page on SourceForge.
