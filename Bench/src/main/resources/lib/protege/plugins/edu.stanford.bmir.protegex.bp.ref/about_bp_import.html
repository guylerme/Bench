BioPortal Import Plugin

The BioPortal Import Plugin allows the user to import classes or entire class trees from external ontologies stored in the BioPortal ontology repository. The BioPortal Import plugin is added as an additional button on the OWL Classes Tab in Protege-OWL.

The basic steps in using the plugin are to:

  1. browse through all the BioPortal ontologies (except remote ones) and select which one to import classes from
  2. select one or more classes to import from the BioPortal ontology
  3. configure the import options (properties and metadata to import, etc.)
  4. import them as subclasses of the selected class in the local ontology.
The plugin uses the RESTful services provided by BioPortal to allow the user to browse through the latest versions of all BioPortal ontologies. The user can view the classes of the selected ontology in an expandable tree format, and also view the properties of each class.

The plugin is configurable and works with Protege-OWL.

For more information, please refer to the BioPortal Import Plugin wiki page.

Download. The plugin is bundled with the Protege installation. However, new versions of the plugin may become available sooner, and they will be published and can be downloaded from the plugin wiki page.
