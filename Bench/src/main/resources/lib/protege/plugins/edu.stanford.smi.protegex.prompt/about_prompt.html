PROMPT Tab

The PROMPT tab allows you to manage multiple ontologies in Protégé, mainly:
  • compare versions of the same ontology
  • move frames between included and including project
  • merge two ontologies into one
  • extract a part of an ontology
For more information, please refer to the Prompt page on the Protégé Wiki.
